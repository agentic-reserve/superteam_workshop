## 📝 Quick Reference

### Common Commands
```bash
# Gateway
openclaw daemon start
openclaw daemon stop
openclaw status

# Channels
openclaw channels add slack
openclaw channels list

# Webhooks
openclaw webhooks create --name "deploy"
openclaw webhooks list

# Skills
openclaw skills reload
openclaw skills list

# Logs
openclaw logs --tail 50
openclaw logs --follow
```

### Configuration Files
```
~/.openclaw/
├── openclaw.json          # Main config
├── AGENTS.md              # Agent instructions
├── skills/                # Skills directory
└── webhooks/              # Webhook handlers
```

### Key Concepts
- **Gateway**: Central runtime for agents
- **Channels**: Platform connections (Slack, Discord, etc)
- **Agents**: AI models with skills and tools
- **Skills**: Specialized knowledge (from workshop)
- **Tools**: Capabilities (exec, browser, web)
- **Webhooks**: HTTP endpoints for events
- **Cron Jobs**: Scheduled tasks
- **Hooks**: Event-driven automation
